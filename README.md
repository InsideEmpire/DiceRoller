# 骰子模拟器

欢迎使用骰子模拟器！这是一个用 Swift 编写的简单而美观的骰子模拟器应用程序，可以模拟不同类型的骰子投掷。

## 功能特点

- 模拟标准六面骰子。
- 可以选择投掷多个骰子。
- 界面简洁、直观，易于使用。

## 如何使用

1. 克隆或下载该仓库到本地计算机。
2. 使用 Xcode 打开 `DiceRoller.xcodeproj` 文件。
3. 在 Xcode 中运行应用程序，选择目标设备或模拟器。
4. 在应用程序中增减骰子数量，然后点击“投掷”按钮进行投掷。
5. 查看投掷结果。

## 技术细节

- 该应用程序使用 Swift 语言编写，基于 iOS 平台。
